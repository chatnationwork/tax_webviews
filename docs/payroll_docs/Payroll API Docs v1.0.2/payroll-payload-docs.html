    Payroll Object

    Overview

    Retrieves detailed payroll information for a specific payroll period, including employee-level breakdowns, tax calculations, and aggregated totals.

    Response Structure

    Top-Level Fields

    Field Type Description
    id integer Unique payroll identifier
    ref_no string Payroll reference number (e.g., "PR-HALLHV")
    period string Payroll period date (ISO 8601 format)
    status string Payroll status (draft, active, etc.)
    type string Payroll type (e.g., regular)
    currency string Currency code (e.g., "KES")
    total_employees integer Total number of employees in payroll
    gross_pay string Total gross pay for all employees
    net_pay string Total net pay for all employees
    gross_tax_due string Total gross tax due before reliefs
    net_tax_due string Total net tax due after reliefs
    filing_date string/null Date payroll was filed with authorities
    payment_date string/null Date employees were paid
    employer_tax_payer_id integer Individual employer's ID
    organization_id integer Organization employer's ID
    inserted_at string Record creation timestamp
    updated_at string Record last update timestamp

    Payroll Items Array

    Contains detailed information for each employee in the payroll.

    Field Type Description
    id integer Unique payroll item identifier
    employee_id integer Employee identifier
    name string Employee full name
    pin string Employee tax PIN
    ref_no string Employee payroll reference
    salary string Base salary amount
    allowance string Total allowances
    benefit string Total taxable benefits
    deduction string Total deductions
    relief string Total tax reliefs
    gross_pay string Employee gross pay
    net_pay string Employee net pay
    taxable_pay string Total taxable income
    tax_due string Total tax payable
    status string Item status
    primary_employee boolean Whether this is a primary employee record
    nita_registered boolean NITA registration status
    email_sent boolean Whether payslip email was sent

    Items Array (Nested in Payroll Items)

    Detailed breakdown of salary components, benefits, deductions, and reliefs for each employee.

    Common Fields:

    • id (string): Unique item identifier (UUID)
    • amount (string): Item amount
    • currency (string): Currency code
    • type (string): Category (salary, allowance, benefit, deduction, relief, employer, exemption, allowable-deduction)
    • subtype (string): Specific item type (e.g., basic-salary, PAYE, NSSF, housing-benefit)
    • status (string): Item status
    • metadata (object): Additional context-specific information

    Common Subtypes by Type:

    Salary: basic-salary, leave-pay

    Allowance: housing-allowance, transport-allowance, overtime-allowance, directors-fee, leave-pay, other-cash-benefits, other-allowance

    Benefit: housing-benefit, car-benefit, other-benefit

    Deduction: PAYE, NSSF, SHIF, AHL, NITA, other, FBT

    Relief: tax-relief, insurance-relief

    Allowable Deduction: pension, PRMF, mortgage-interest, meal_benefit, non_cash_benefit

    Employer: NSSF, AHL, NITA, FBT

    Exemption: disability-exemption

    Tax Aggregates Array

    Summarized totals for each tax component across all employees.

    Field Type Description
    id string Unique aggregate identifier (UUID)
    type string Component category
    subtype string Specific component type
    amount string Total amount for this component
    currency string Currency code
    quantity integer Number of employees with this component
